Piano

Watch Demo Video

Developed a modular electronic keyboard system based on STM32 and FreeRTOS, where each keyboard module contained seven keys and up to four modules could be cascaded to form a complete instrument. The modules communicated and coordinated via a CAN bus, enabling scalable system integration. The system supported configurable parameters such as pitch, timbre, and sustain, demonstrating embedded system design capabilities in real-time multitasking and inter-module communication.

FPGARUSH

Watch Demo Video

Developed a networked racing game system based on FPGA and Unity, where the FPGA served as a hardware controller responsible for vehicle acceleration, braking, and steering, while Unity handled game logic and graphical rendering. The system synchronized game state data with a remote server to enable multi-client online multiplayer racing, demonstrating real-time interaction and hardware–software co-design capabilities.
